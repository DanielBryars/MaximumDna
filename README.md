# MaximumDna

A web-based audio file hosting repository for BBC Radio 4 Shipping Forecast recordings.

## Overview

This repository hosts audio files of the BBC Radio 4 Shipping Forecast in μ-law encoded format at 8kHz sampling rate, accessible via a simple HTML index page.

## Contents

- **ShippingForecast.1stMin.ulaw.8k.wav**: First minute of the Shipping Forecast
- **ShippingForecast.Full.ulaw.8k.wav**: Full Shipping Forecast broadcast
- **index.html**: Simple directory listing for audio file access

## Audio Format

- **Encoding**: μ-law (mu-law) - logarithmic compression standard for telephony
- **Sample Rate**: 8 kHz - standard for telephone-quality audio
- **Format**: WAV container

## About the Shipping Forecast

The Shipping Forecast is a BBC Radio 4 broadcast providing weather reports and forecasts for the seas around the British Isles. It's a British institution, known for its distinctive format and calming delivery.

## Usage

Open `index.html` in a web browser to access the audio files through clickable links.
